## What Was Amplified

The tweet summarized all 5 Token-2022 extensions from the Week 6 build log:

| Extension | Key Rule |
|-----------|----------|
| Transfer Fees | Auto-enforced % on every transfer |
| Interest-Bearing | Calculation layer, not new token supply |
| Default Frozen | Both sender AND recipient need to be thawed |
| Non-Transferable | `0x25` error — no workaround at protocol level |
| Permanent Delegate | Issuer burns without holder's signature |
